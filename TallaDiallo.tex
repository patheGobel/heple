($f$ est une fonction numérique d'une variable réelle, $\ell$ un réel)
\begin{enumerate}
\item \[\lim_{x \to -\infty} f(x) = l\]

\item \[\lim_{x \to +\infty} f(x) = +\infty\]

\item \[\lim_{x \to -\infty} f(x) = +\infty\]

\item \[\lim_{x \to -2} \ln(2x+5) = 0\]

\end{enumerate}
